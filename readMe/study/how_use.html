工具使用.
1、在插件安装成功之后，任意选中一个java文件用Easy Code编辑器方式打开，在第一次打开文件的过程中，系统自动会在项目中生成一个GenConfig文件夹(在eclipse中刷新项目后可看到)， 一般情况下，这个文件夹不用理会。
GenConfig文件夹:
该文件夹中有一个文件GenInitConfig.xml,里面有些默认的配置信息，包括本地模板的存放路径，模板服务器信息(如果模板都存放在本地，则不需要设置)， 等等。 具体配置信息可以通过【Easy Code 配置】编辑器进行查看设置。
2、提供了多种使用方式，依照使用习惯自己选择。
使用方式一，通过Easy Code编辑器方式打开，通过该方式可以方便的给类，属性或方法添加注释。
使用方式二，采用eclipse默认的编辑器方式打开，点击工具栏上的Easy Code图标。
使用方式三，选中一个java文件，右键菜单中选中Easy Code
