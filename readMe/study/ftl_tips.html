提示框标签.
提示框信息包含两种，警告提示，错误提示。
1、警告提示框:<ftl_warn></ftl_warn>
   在预览或代码生产过程中,弹出该标签,即提示用户,是否继续执行.点击是继续执行,否就停止执行.
2、错误提示框:<ftl_err></ftl_err>
   模板解析过程中，如果没达到模板所必备的条件，则弹出此框，弹出此框，则文档会停止执行或预览。
